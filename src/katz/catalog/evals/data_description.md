# Data Description

Is the dataset adequately described? The reader should understand where the data come from, how they were collected, and the basic shape of the sample. Consider:

- What is the source of the data (survey, administrative records, scraped, experimental)?
- How was the data collected, and over what time period?
- How many observations are there before and after any sample restrictions?
- What data were dropped, and is the rationale for each exclusion clearly justified?
- Is there a summary statistics table covering all variables used in the analysis?
- Is every number in a summary statistics table described or referenced in the text?
- Are cross-tabulations shown for categorical variables? Is the distribution of continuous variables shown (histograms, kernel densities, or at minimum percentiles)?

A reader should be able to assess data quality and generalizability from the data description alone, without needing to consult an appendix or replication package.
